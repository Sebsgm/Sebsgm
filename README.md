## 🚀 Proyectos Destacados

### 🌐 [DevTree](https://github.com/Sebsgm/DevTree) — FullStack Practice Project
> Proyecto FullStack con arquitectura separada Backend/Frontend. Desarrollado en **TypeScript (97.3%)**, con estructura moderna de carpetas. Exploración práctica de tecnologías web actuales con manejo de rutas, componentes y API.

**Stack:** TypeScript · JavaScript · Node.js · React

---

### 📅 [TodoUni](https://github.com/Sebsgm/TodoUni) — Gestor Universitario
> Aplicación web **local-first** para estudiantes universitarios. Permite crear horarios semanales visuales con bloques proporcionales por materia, exportarlos a PDF y hacer seguimiento de notas por cortes de evaluación. Sin backend ni dependencias externas — todo corre en el navegador.

**Funcionalidades:**
- Constructor de horarios semanal (lunes a sábado, resolución de 30 minutos)
- Exportación a PDF en formato A4 horizontal con leyenda de materias
- Calculadora de notas con proyección de puntaje necesario para aprobar
- Soporte para múltiples semestres independientes
- Modo oscuro con CSS custom properties
- Almacenamiento en `localStorage` — sin registro ni internet

**Stack:** React 18 · Vite 5 · JavaScript · CSS · jsPDF

### 🔐 [Proyecto Práctica de Ingeniería](https://github.com/Sebsgm/Proyecto---Practica-de-Ingenieria) — Ciberseguridad para PYMEs *(En progreso)*

**Stack:** TypeScript (98.6%) · SQL · Node.js · React

### 🏪 [Proyecto TecnoSuministros](https://github.com/Sebsgm/Proyecto-TecnoSuministros) — Sistema de Inventarios

**Stack:** Java · Java Swing (JFrame) · MySQL · SCRUM

---

### 💼 [Proyecto Nómina](https://github.com/Sebsgm/Proyecto-Nomina) — Gestión de Nómina
